# form-person-loan

## Descripción
Componente de formulario para registrar préstamos de equipos a personas. Permite capturar la información necesaria para realizar un préstamo de inventario, incluyendo datos personales del solicitante y detalles del préstamo.

## Características
- Formulario reactivo con validaciones
- Campos obligatorios y opcionales claramente marcados
- Validación en tiempo real
- Diseño responsive
- Integración con servicios de préstamos
- Mensajes de éxito y error

## Uso
Se utliza exclusivamente para la creacion de un prestamo, 

## Parametros input
- Se le debe pasar el id del item al que se le quiere hacer el prestamo 
- Utiliza un validador personalizado, se importa de la carpeta de app/validators para la validacion de la fecha
