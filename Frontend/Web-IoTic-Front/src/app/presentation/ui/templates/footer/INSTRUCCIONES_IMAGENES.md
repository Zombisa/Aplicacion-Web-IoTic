# Instrucciones para Añadir Imágenes al Footer

## Ubicación de las Imágenes

Las imágenes de los logos deben colocarse en la carpeta:
```
src/assets/logos/
```

## Nombres de Archivos Requeridos

1. **universidad-cauca.png** - Logo de la Universidad del Cauca
2. **gov-co.png** - Logo de gov.co
3. **icontec-iso.png** - Logo de icontec ISO 9001
4. **iqnet.png** - Logo de IQNET

## Pasos para Añadir las Imágenes

1. Crea la carpeta `logos` dentro de `src/assets/` si no existe:
   ```
   src/assets/logos/
   ```

2. Coloca las imágenes en esa carpeta con los nombres exactos mencionados arriba.

3. Las imágenes se cargarán automáticamente. Si alguna imagen no se encuentra, se mostrará un espacio en blanco.

## Formato Recomendado

- **Formato**: PNG con fondo transparente (preferible) o JPG
- **Tamaño**: Aproximadamente 100x70px (se ajustarán automáticamente)
- **Peso**: Optimizar las imágenes para web (menos de 100KB cada una)

## Verificación

Una vez añadidas las imágenes, verifica que se muestren correctamente en el footer de la aplicación.
