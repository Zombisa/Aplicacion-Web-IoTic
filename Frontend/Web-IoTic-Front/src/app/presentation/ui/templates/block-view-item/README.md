# block-view-item

## Descripción
Componente para visualizar el item dentro de un bloque, maneja diseño responsive, no edita , ni realiza ninguna accion
## Características
 -responsive
## Parametros Input
-utliza un input de itemId, debe ser mandado desde el padre para que el componetne sea el encargado de consultar el item y mostrarlo en pantalla
## Uso
Se utliza exclusivamente para la visualizacion de un item dentro de una columna
