# view-item

## Descripción
Pagina para la visualizacion de la informacion detallada del item 
## Reglas de negocio:
-No permitir la opcion de dar en caldiad de presta a objetos dañados
-No permitir la opcion de dar en caldiad de presta a objetos en estado administrativo ="Prestado"
-No permitir la opcion de dar en caldiad de presta a objetos en estado administrativo = "No prestar
## Características
 -responsive
## Parametros url
-utliza una consulta al a url para el  itemId, debe ser mandado desde la pagina anterior por medio de la navegacion  para que el componetne sea el encargado de consultar el item y mostrarlo en pantalla
## Uso
Se utliza exclusivamente para la visualizacion de un item en pantalla completa y con toda la informacion especificada
