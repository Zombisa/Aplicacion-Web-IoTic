# add-loan

## Descripción
Pagina para la gestionar el tramite de la peticion de un item en calidad de prestamo

## hijos
-block-view-item para la visualizacion de la informacion del item 
-form-person-loan para realizar la diligenciacion del formulario del preastamo del item
## Características
 -responsive
 -utliza componentes hijos
## Parametros url
-utliza una consulta al a url para el  itemId, debe ser mandado desde la pagina anterior por medio de la navegacion  para que el componetne sea el encargado de redigir el itemID a sus componentes hijos encargados cada uno de su funcion especifica
## Uso
Se utliza exclusivamente para la gestion del item en calidad de prestamo
